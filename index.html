Project: Meet App

The Meet App is a progressive web application (PWA) that allows users to find tech events in their city. It includes dynamic data visualizations, offline support, and filtering functionality to enhance the user experience. Built with a test-driven development (TDD) approach.

Key Features

  • Google Calendar OAuth integration
  • Interactive Recharts Pie and Scatterplots
  • Offline support via service workers
  • Test Driven Development (TDD)

Technologies Used

React.js • Recharts • Jest • Cucumber • AWS Lambda

Project: Chat App (Mobile)

A native mobile chat application built for Android and iOS. Users can enter chat rooms, share images from their gallery or camera, and share their real-time location. The app features robust offline capabilities, allowing users to read cached messages even without an internet connection.

Key Features

  • Built with React Native & Expo
  • Real-time chat via Google Firestore
  • Image sharing (Camera/Library) & Geolocation features
  • Offline data caching with AsyncStorage
  • Accessibility features (Screen reader support)

Technologies Used

React Native • Expo • Google Firestore • Firebase Storage
